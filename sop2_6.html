1. Inicjalizacja Pamięci (Wzorzec "Otwórz lub Stwórz")

Używamy O_EXCL, aby wiedzieć, czy proces jest "Pierwszy" (inicjator).
// Struktura dzielona
typedef struct {
    pthread_mutex_t mutex;
    int counter;
    int end_flag;
} shared_t;

shared_t* setup_shm(const char* name, int* is_first) {
    int fd = shm_open(name, O_CREAT | O_EXCL | O_RDWR, 0666);
    *is_first = 1;

    if (fd == -1 && errno == EEXIST) {
        fd = shm_open(name, O_RDWR, 0666);
        *is_first = 0;
    } else if (fd == -1) { perror("shm_open"); exit(1); }

    if (*is_first) ftruncate(fd, sizeof(shared_t));

    shared_t* ptr = mmap(NULL, sizeof(shared_t), PROT_READ | PROT_WRITE, MAP_SHARED, fd, 0);
    close(fd);
    return ptr;
}

2. Mapowanie pliku log.txt (Bitronka)

Pozwala modyfikować plik jak tablicę znaków char[].
char* map_log_file(const char* path, size_t size) {
    int fd = open(path, O_RDWR | O_CREAT | O_TRUNC, 0666);
    ftruncate(fd, size);
    char* ptr = mmap(NULL, size, PROT_READ | PROT_WRITE, MAP_SHARED, fd, 0);
    close(fd);
    return ptr;
}

3. Mutexy: Inicjalizacja i Robustness

Kluczowe: PTHREAD_PROCESS_SHARED i obsługa EOWNERDEAD.
void init_mutex(pthread_mutex_t* m) {
    pthread_mutexattr_t attr;
    pthread_mutexattr_init(&attr);
    pthread_mutexattr_setpshared(&attr, PTHREAD_PROCESS_SHARED);
    pthread_mutexattr_setrobust(&attr, PTHREAD_MUTEX_ROBUST);
    pthread_mutex_init(m, &attr);
}

void safe_lock(pthread_mutex_t* m) {
    if (pthread_mutex_lock(m) == EOWNERDEAD) {
        pthread_mutex_consistent(m);
    }
}

4. Obsługa Sygnałów i "Czyste Wyjście"

Zapobiega ubiciu procesu w trakcie ważnych obliczeń.
volatile sig_atomic_t keep_running = 1;

void sig_handler(int sig) {
    if (sig == SIGINT) keep_running = 0;
}

// W main:
struct sigaction sa = {.sa_handler = sig_handler};
sigaction(SIGINT, &sa, NULL);

5. Cykl Życia: Tworzenie i Sprzątanie

void create_workers(int n, shared_t* sh) {
    for (int i = 0; i < n; i++) {
        if (fork() == 0) {
            worker_logic(i, sh);
            exit(0); // Dziecko musi tu umrzeć
        }
    }
    while (wait(NULL) > 0); // Czekanie na wszystkich
}

6. Precyzyjne Usypianie (ms_sleep)

void ms_sleep(int milis) {
    struct timespec ts = {milis / 1000, (milis % 1000) * 1000000L};
    nanosleep(&ts, NULL);
}

7. Algorytm rozwiązywania zadań (Workflow)

Lista kroków do odhaczenia podczas kolokwium:
  • Etap 1: Zdefiniuj strukturę w pamięci dzielonej. Użyj shm_open z O_EXCL.
  • Etap 2: Zaimplementuj fork(). Pamiętaj o exit(0) w dziecku.
  • Etap 3: Dodaj sigaction dla SIGINT. Nie używaj przestarzałego signal().
  • Etap 4: Obsłuż EOWNERDEAD przy blokowaniu mutexa.

8. Słownik Pułapek

  • Bus Error Brak ftruncate.
  • Zombie Brak wait().
  • Deadlock Brak ROBUST przy padzie procesu.
