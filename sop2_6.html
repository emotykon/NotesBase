🐧 SOP: Kompendium dla Opornych (Linux IPC)

Ściągawka do zadań z systemów operacyjnych (Bitronka, Całkowanie, Pamięć Dzielona).

Zasada Złota: Zawsze sprawdzaj błędy funkcji systemowych (if (result == -1)). W SOP błędy nie znikają same!

1. Nagłówki i Setup

#define _GNU_SOURCE
#include <stdio.h>
#include <stdlib.h>
#include <unistd.h>     // fork, ftruncate, usleep
#include <sys/mman.h>   // mmap, shm_open, shm_unlink
#include <sys/stat.h>   // uprawnienia (0666)
#include <fcntl.h>      // O_CREAT, O_RDWR
#include <pthread.h>    // Mutexy, bariery
#include <signal.h>     // SIGINT, SIGKILL
#include <errno.h>      // EOWNERDEAD

2. Pamięć Współdzielona (SHM)

Schemat: shm_open (stworzenie) → ftruncate (rozmiar) → mmap (dostęp).

// 1. Otwieranie obiektu
int fd = shm_open("/moje_shm", O_CREAT | O_RDWR, 0666);
if (fd == -1) perror("shm_open");

// 2. Nadawanie rozmiaru (KRYTYCZNE!)
ftruncate(fd, sizeof(shared_data_t));

// 3. Mapowanie do struktury
shared_data_t *data = mmap(NULL, sizeof(shared_data_t), 
                           PROT_READ | PROT_WRITE, MAP_SHARED, fd, 0);
close(fd); // Deskryptor nie jest już potrzebny po mmap

3. Robust Mutex

Niezbędny, gdy procesy mogą "zginąć" (np. przez abort()).

Krok Funkcja
Atrybuty pthread_mutexattr_setrobust(&attr, PTHREAD_MUTEX_ROBUST);
Blokada if (pthread_mutex_lock(&m) == EOWNERDEAD) pthread_mutex_consistent(&m);

4. Zarządzanie Procesami (Fork)

for (int i = 0; i < num_workers; i++) {
    switch (fork()) {
        case -1: perror("fork"); exit(1);
        case 0:  
            // Kod dziecka
            printf("Pracownik %d startuje\n", getpid());
            do_work(i, data);
            exit(EXIT_SUCCESS); // Dziecko musi umrzeć tutaj!
        default: break; // Rodzic idzie dalej tworzyć dzieci
    }
}
// Rodzic czeka na koniec
while (wait(NULL) > 0);

5. Obsługa Sygnałów

Służy do bezpiecznego kończenia pętli while(1).

volatile sig_atomic_t keep_running = 1;
void handler(int sig) { keep_running = 0; }

// W main:
struct sigaction sa = {.sa_handler = handler};
sigaction(SIGINT, &sa, NULL);

// W pętli procesu:
while(keep_running) { ... }

6. Słownik Pułapek

  • Bus Error Zapomniałeś o ftruncate.
  • Zombie Zapomniałeś o wait() lub waitpid().
  • Deadlock Proces padł trzymając zwykły mutex (użyj Robust!).
  • Leaking SHM Zapomniałeś o shm_unlink() na końcu serwera.
