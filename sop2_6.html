🐧 SOP: Kompendium dla Opornych (Linux IPC)

Zoptymalizowana ściągawka do zadań z laboratoriów. Używaj przycisków Copy do szybkiego wklejania szablonów.

Zasada Złota: Pamiętaj o kompilacji z flagami: -lpthread -lrt. Bez tego funkcje SHM i Mutexy nie zadziałają!

1. Nagłówki i Setup

#define _GNU_SOURCE
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <unistd.h>     // fork, ftruncate, usleep
#include <sys/mman.h>   // mmap, shm_open, shm_unlink
#include <sys/stat.h>   // uprawnienia (0666)
#include <sys/wait.h>   // wait
#include <fcntl.h>      // O_CREAT, O_RDWR
#include <pthread.h>    // Mutexy
#include <semaphore.h>  // Semafory nazwane
#include <signal.h>     // SIGINT, SIGKILL
#include <errno.h>      // EOWNERDEAD

2. Pamięć Współdzielona (SHM)

// 1. Definicja struktury (Dopasuj do zadania)
typedef struct {
    pthread_mutex_t mutex;
    int data[256];
    int counter;
    int stop_flag;
} shared_t;

// 2. Setup (Tylko Rodzic/Serwer)
int fd = shm_open("/my_shm", O_CREAT | O_RDWR, 0666);
ftruncate(fd, sizeof(shared_t));
shared_t *sh = mmap(NULL, sizeof(shared_t), PROT_READ | PROT_WRITE, MAP_SHARED, fd, 0);
close(fd);

3. Bitronka: Mutexy na każdą półkę

Jeśli zadanie wymaga blokowania konkretnych indeksów tablicy:

// W strukturze:
pthread_mutex_t locks[256]; 

// Inicjalizacja atrybutów:
pthread_mutexattr_t attr;
pthread_mutexattr_init(&attr);
pthread_mutexattr_setpshared(&attr, PTHREAD_PROCESS_SHARED);
pthread_mutexattr_setrobust(&attr, PTHREAD_MUTEX_ROBUST);

for(int i = 0; i < N; i++) {
    pthread_mutex_init(&sh->locks[i], &attr);
}

4. Obsługa Śmierci (Robust Mutex)

Kluczowe dla etapu 4 Bitronki (obsługa abort()):

int res = pthread_mutex_lock(&sh->locks[idx]);
if (res == EOWNERDEAD) {
    printf("[%d] Found a dead body at aisle %d\n", getpid(), idx);
    pthread_mutex_consistent(&sh->locks[idx]);
    // Tu opcjonalnie: sh->workers_alive--;
}
// ... praca ...
pthread_mutex_unlock(&sh->locks[idx]);

5. Całkowanie: Semafor Nazwany

Zapobiega wyścigowi przy jednoczesnym starcie wielu procesów:

sem_t *gate = sem_open("/shm_gate", O_CREAT, 0666, 1);
sem_wait(gate); 

// Sekcja krytyczna (np. pierwsza inicjalizacja SHM)
// if (sh->counter == 0) { ... }

sem_post(gate);
sem_close(gate);
// Na samym końcu programu: sem_unlink("/shm_gate");

6. Obsługa Sygnałów (SIGINT)

volatile sig_atomic_t keep_running = 1;
void handler(int sig) { keep_running = 0; }

// W main():
struct sigaction sa = {.sa_handler = handler};
sigaction(SIGINT, &sa, NULL);

// W pętli:
while(keep_running && !sh->stop_flag) { ... }

7. Słownik Pułapek

  • Bus Error Brak ftruncate lub zły rozmiar mmap.
  • Zombie Zapomniałeś o while(wait(NULL) > 0); u rodzica.
  • 404 na GitHubie Poczekaj minutę na zielony ptaszek w zakładce Actions.
  • Mutex nie działa Sprawdź czy ustawiłeś PTHREAD_PROCESS_SHARED!
